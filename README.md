# gratuation
毕业设计
## 大学毕业设计，设计一款摄影公司的页面。

## 简述:
摄影公司简介，包含了摄影公司业务范围，硬件设备。人才储备、作品展示、服务信息、 网上预订、留言 信息。 
## UI 设计:
https://lanhuapp.com/url/mMIwJ 
## 实现方法:
用 gulp webapp(npm)来搭建项网站，其中 bootstrap 实现响应式页面，scss 改变样式，jq 实现更 多的交互页面。
